# 金刚经配图版

> 原文《金刚经》提炼每一品最能代表主题的核心句整理，并配以 16:9 水墨佛学风格插图。  
> An illustrated reader of the Diamond Sutra, with selected core lines from each chapter and AI-generated visual cards.

作者：@文韬推荐

## 这是什么

《金刚经配图版》是一个经典阅读实验：把《金刚经》三十二品整理为图文卡片，每品提炼 2 条代表性经文，配合白话理解和水墨风格插图，降低第一次阅读的门槛。

## 当前内容

- 32 品目录导航
- 64 条核心经文卡片
- 64 张 16:9 配图
- 白话提示和灯箱阅读
- 诵读音频播放入口

## 必须说明

本项目不是权威佛学注疏，也不替代正式译注。经文摘句和白话解释偏向普通读者阅读体验。

配图由 AI 生成，可能存在意象不准确、风格不一致等问题。欢迎基于原文继续校订、替换配图或改进阅读体验。

## 快速开始

```bash
npm install
npm run dev
```

构建静态页面：

```bash
npm run build
```

## GitHub Pages

如果部署到 GitHub Pages，并且仓库名不是根域名仓库，可以在构建时设置：

```bash
PUBLIC_BASE_PATH=/jingangjing-illustrated-reader/ npm run build
```
